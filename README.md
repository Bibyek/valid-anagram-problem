# Valid Anagram (Java)

## Problem
Given two strings, determine whether they are anagrams of each other.

## Definition
An anagram is a word formed by rearranging the letters of another word using all original letters exactly once.

## Example
Input: "listen", "silent"  
Output: true

Input: "rat", "car"  
Output: false

## Approach
- Check if lengths are equal
- Use a frequency array to count characters
- Compare final counts

## Time Complexity
O(n)

## Space Complexity
O(1)

## Language
Java
